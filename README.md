# -AYDS1_G3
Este proyecto aborda el problema de la gestión ineficiente de citas médicas en clínicas que carecen de plataformas digitales integradas. La solución propuesta es SaludPlus, una plataforma web que facilita la comunicación entre pacientes y médicos, permitiendo la reserva, gestión y seguimiento de citas médicas de forma eficiente.
