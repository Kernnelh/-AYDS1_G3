# SaludPlus - Plataforma de Gestión de Citas Médicas

 **SaludPlus** es un sistema diseñado para facilitar la conexión entre pacientes y médicos, permitiendo la gestión eficiente de horarios, programación de citas y expedientes médicos básicos.

* [Módulos](#Módulos)
* [Documentación](#Documentación)
* [Tecnologías](#Tecnologías)

---

## Módulos
- **Frontend:** Interfaz de usuario intuitiva (Ver carpeta `/frontend`).
- **Backend:** API REST robusta para la gestión del negocio (Ver carpeta `/backend`).
- **Database:** Base de datos relacional MySQL (Ver carpeta `/database`).
